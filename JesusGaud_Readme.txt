Project 2 Part 1 - Cluedo Game

Student Name:
Jesus Gaud

Description:
This project implements the first part of a Cluedo-style game in Python. The application includes a mansion layout, character definitions, weapon definitions, random solution selection, player movement between rooms, and the ability to make suggestions.

Files Included:
main.py
game.py
mansion.py
character.py
weapon.py
player.py
solution.py
JesusGaud_Readme.txt

How to Run:
1. Download or unzip the project folder.
2. Open a terminal or command prompt.
3. Navigate to the project folder:

   cd JesusGaud_Project2_SourceCode

4. Run the program:

   python main.py

Requirements:
Python 3.x

Game Features:
- Mansion layout with connected rooms
- Character definitions and starting rooms
- Weapon definitions
- Random solution selection
- Player movement between adjacent rooms
- Suggestion feature using character, weapon, and current room

Notes:
This project focuses on Part 1 requirements: game setup and player movement. Additional features can be added in Part 2.
